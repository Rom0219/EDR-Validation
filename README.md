# Teoría del Espacio Dinámico Rotacional (EDR)
Validación Numérica Oficial  
Autor: **Camilo Robinson**

Este repositorio contiene:

- Implementación numérica del potencial EDR δV(r)
- Cálculo del corrimiento de frecuencia δω
- Predicción del parámetro α_flow
- Estimación de la frecuencia del modo X para agujeros negros tipo Kerr
- Scripts de validación de curvas de rotación (SPARC)
- Documentación teórica completa en `/docs`
